## APP特點
* 直接由Google帳號驗證登入
* 新增代辦家務、認領家務
* 與Firestore串聯，即時上傳APP使用者填寫資料
* 其他裝置登入相同帳號者，可以即時更新在其他裝置中
* 介面簡單整齊，滾輪調整時間，按鈕自訂喜好程度
* 每周數據報表呈現    

#### 已開發功能
* 支援Google帳號登入
* App 使用者輸入紀錄即時上傳Firestore
* 偵測使用者是否輸入完整
* AppBar頁面切換

#### 待開發
* Firestore資料即時回傳
* 紀錄更新可以讓Unidust灰塵寶寶升級，並同時回傳至app
